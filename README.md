# study-typescript-01

## install
- VSCode
- Node.js

## TypeScript install
### `npm install -g typescript`

### setting
- index.ts (ts: 타입스크립트 파일)
- tsconfig.json 생성
- tsconfig.json 에 내용작성 --> {"compilerOptions": {"target": "ES6", "module": "CommonJS"}}

## TypeScript start
### `tsc -w`

## React Project
### `npm install --save typescript @types/node @types/react @types/react-dom @types/jest`
